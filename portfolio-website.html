Personal Portfolio Website

Deze website werd volledig zelf ontwikkeld op basis van de Ceevee template. Het doel was om een moderne, recruiter-vriendelijke portfolio te bouwen die mijn profiel als Support Engineer duidelijk weergeeft en mijn groei richting System, Network en Cloud Engineer ondersteunt.

Technologieën

  • HTML5
  • CSS3
  • JavaScript
  • Formspree (contactformulier)

Wat ik heb gebouwd

  • Volledig responsive website (mobile-first)
  • Aangepaste Ceevee layout
  • Accordion systeem voor werkervaring
  • Portfolio sectie met uitbreidbare projecten
  • Werkend contactformulier zonder backend

Wat ik geleerd heb

  • Structureren van een professionele website
  • UX/UI optimalisatie voor recruiters
  • JavaScript interacties (accordion)
  • Deployment-ready website bouwen

Toekomstige uitbreidingen

  • Integratie van Azure en netwerk projecten
  • Project screenshots en diagrammen
  • Meer interactieve elementen
